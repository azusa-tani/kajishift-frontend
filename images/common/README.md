# 共通画像フォルダ

このフォルダには、全てのページ（customer、worker、admin）で使用できる共通画像を配置してください。

## 推奨される画像の種類

### アイコン・ロゴ
- サイトロゴ
- ファビコン
- 共通アイコン（通知、設定、ヘルプなど）

### プレースホルダー画像
- デフォルトアバター
- 画像未設定時のプレースホルダー

### 共通UI要素
- ボタン背景
- 装飾用のパターンやグラデーション

## 使用方法

HTMLファイルからは以下のように参照してください：

```html
<!-- 相対パス（customer/worker/adminフォルダ内のHTMLから） -->
<img src="../images/common/logo.png" alt="ロゴ">

<!-- 絶対パス（index.htmlから） -->
<img src="images/common/logo.png" alt="ロゴ">
```

## ファイル命名規則

- 小文字とハイフンを使用（例: `site-logo.png`）
- 用途が分かりやすい名前を付ける
- 拡張子は適切な形式を使用（.png, .jpg, .svg など）
